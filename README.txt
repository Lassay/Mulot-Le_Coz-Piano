Le projet global se lance avec piano.py (Lors du premier lancement, les notes mettent un peu de temps à ce générer)

Ce programme lance dans une même interface trois programmes indépendants :

 - wave_generator.py génère l'interface de création de notes et d'accords. Nous pouvons génerer la note ou l'accord de notre choix en choisissant la tonalité, l'octave et la durée pour la note pur. Ce qui est créé est ensuite stocké dans la liste correspondante (note ou accord). Nous devons ensuite cliquer sur ce que nous voulons jouer puis cliquer sur lire pour entendre notre création.

 - frequencies_MVC.py génère la visualisation du signal de la note joué.

 - keyboard.py génère les touches de piano. En cliquant, nous pouvons voir le nom de la note jouée au dessus du clavier.
